To apply seeds for system-service locally:

1. Ensure DATABASE_URL points to your system_db (e.g., export DATABASE_URL="postgresql://stackline:stackline@localhost:5432/system_db")
2. Run prisma migrate or create database/tables if not present.
3. Apply seed SQL:
   psql "$DATABASE_URL" -f prisma/seed.sql

Note: The seed.sql assumes table names generated by Prisma (package, microfrontend, microfrontend_access). If your database schema differs, adapt the SQL.
